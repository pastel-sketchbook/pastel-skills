# FFE Light Theme

Use this palette for all decks:

- Background: `#edf0f4`
- Foreground: `#1e232b`
- Selection: `#dde1e8`
- Red: `#c9434e`
- Green: `#1a8a6e`
- Yellow: `#b8920d`
- Blue: `#3a8ea4`
- Magenta: `#c0587a`
- Cyan: `#2a9d84`
- Muted: `#7c8496`
- Panel: `#ffffff`

Presentation direction: light editorial deck, crisp terminal-inspired code panels, strong contrast, no dark theme.

Use IBM Plex fonts: IBM Plex Sans for UI/body/headings, IBM Plex Serif for title-slide display accents, and IBM Plex Mono for code. If unavailable, use close system fallbacks but keep the same roles.

Code panels should feel filled and intentional: include a compact header with file path, show 12-28 readable lines, use complete code blocks or self-contained excerpts with braces/expressions closed, avoid placeholder ellipses, and highlight the lines that explain the critical point.
